# PhantomJS安装

标签（空格分隔）： PhantomJS

---


### 1.下载包
```bash
wget https://bitbucket.org/ariya/phantomjs/downloads/phantomjs-1.9.8-linux-x86_64.tar.bz2
```
 
### 2.解压
```bash
tar jxf phantomjs-1.9.8-linux-x86_64.tar.bz2
```

### 3.创建链接
```bash
ln -s phantomjs-1.9.8-linux-x86_64/bin phantomjs
```
